#Hospital Patient Data Analysis

This project analyzes hospital patient data using Python.

Files:
- analysis.py : Reads and merges the CSV files and performs analysis
- dashboard.py : Creates the dashboard and charts
- data/ : Contains patients, doctors and appointments CSV files
- output/ : Contains processed results and charts

Features:
- Number of patients per doctor
- Most common diseases
- Dashboard visualizations

Tools Used:
- Python
- Pandas
- Plotly
- Dash
- SQLite
